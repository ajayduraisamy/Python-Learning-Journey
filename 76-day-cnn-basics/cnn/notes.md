# CNN Basics (DL Day 5)

### Why CNN?
CNNs are used for images because:
- They capture spatial patterns
- They understand edges, shapes, textures
- They use fewer parameters than dense layers

### Key Concepts:
1. **Convolution Layer**
   - Applies filters (kernels)
   - Detects edges / patterns

2. **Stride**
   - Moves the filter across the image

3. **Padding**
   - SAME: output size preserved
   - VALID: no padding

4. **Max Pooling**
   - Reduces size
   - Keeps strongest features

5. **Flatten + Dense**
   - Final classification
